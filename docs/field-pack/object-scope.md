# Object Scope Field Pack

Generic object-level scope identifiers.

## Canonical columns

- `object_scope`
- `object_tenant`
- `object_organization`
- `object_owner`

## Boundary

Object scope is not RBAC or authorization. Security policy remains in backend/security components.

## Package shape

Each pack has an embeddable, embeddable trait, mirror interface, YAML manifest, registry entry, and quality gate coverage.
