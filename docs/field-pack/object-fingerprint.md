# Object Fingerprint Field Pack

Generic technical fingerprint/integrity surface.

## Canonical columns

- `object_hash`
- `object_checksum`
- `object_algorithm`

## Boundary

Object fingerprint is not idempotency, attachment storage, or cryptographic signing policy.

## Package shape

Each pack has an embeddable, embeddable trait, mirror interface, YAML manifest, registry entry, and quality gate coverage.
