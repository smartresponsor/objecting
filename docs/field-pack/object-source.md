# Object Source Field Pack

Generic source/origin and external reference surface.

## Canonical columns

- `object_source`
- `object_provider`
- `object_external_id`
- `object_source_type`

## Boundary

Object source is not an integration framework. Business provider logic stays in the backend component.

## Package shape

Each pack has an embeddable, embeddable trait, mirror interface, YAML manifest, registry entry, and quality gate coverage.
